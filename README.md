# Together Websites

Monorepo containing three websites for Together Assessments, a neurodiversity assessment service provider:

- **Assessments** - Main service website
- **ADHD** - ADHD-focused information and services
- **Autism** - Autism-focused information and services

## Documentation

Technical documentation is available in the `technical-docs/` folder:

- **WEBSITE.md** - Site structure and content
- **ARCHITECTURE.md** - Technical architecture and configuration
- **CHEATSHEET.md** - Component reference
- **CMS.md** - Content management system
- **COLOURS.md** - Colour system and theming
- **ACCESSIBILITY.md** - Accessibility features

Developer guidance is in `CLAUDE.md`.

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS](https://tailwindcss.com/) - Styling
- [Decap CMS](https://decapcms.org/) - Content management

Based on the [AstroWind](https://github.com/onwidget/astrowind) template.
